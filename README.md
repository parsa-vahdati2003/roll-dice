# Roll Dice

A simple project that simulates the rolling of a dice. This project is built using JavaScript and HTML, providing a fun way to simulate dice rolls for games or other purposes.

## Features
- Simulates the rolling of a dice (1 to 6).
- Displays the rolled number with a visual representation.
- Easy to modify and extend.
